# Cofre técnico do ChatPro

Este repositório é o Cofre técnico externo do ChatPro. Ele funciona como memória operacional para o Codex, concentrando contexto, decisões, manutenção e histórico sem misturá-los ao código da aplicação.

O projeto principal e o Cofre são repositórios Git independentes. A documentação deve registrar caminhos reais, dependências e impactos cruzados, sem presumir a existência de código ainda não criado.

O Cofre deve permanecer sincronizado entre os dois computadores de desenvolvimento. No início de cada tarefa, consulte primeiro o [INDEX.md](INDEX.md), que direciona aos documentos relevantes. Não percorra todo o Cofre indiscriminadamente.
